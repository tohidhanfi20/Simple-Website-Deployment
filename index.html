Welcome to my website hosted on the cloud using Terraform and Route 53!

About This Project

This website demonstrates how to deploy a simple website using Terraform and AWS services. The site is mapped to a custom domain name for easy access.

Technologies Used

  • Terraform for Infrastructure as Code
  • AWS EC2 for hosting
  • Route 53 for domain mapping
  • HTML and CSS for the website
